## Features
- **User Typed:** Send your own custom messages repeatedly.
- **Counting Mode:** Automatically adds numbers to your messages (e.g., "Hello 1", "Hello 2").
- **Random Mode:** Generates messages using a list of 10.000 English words.
- **Meaningless Mode:** Sends random strings of characters and numbers.
